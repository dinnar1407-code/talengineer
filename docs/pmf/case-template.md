## 1. 客户与现场（脱敏）

- **行业 / 板块**：（例：汽车零部件、食品饮料、光伏——不写公司名）

- **国家 / 地区**：（脱敏到国家或大区，不精确到城市/园区）
- **雇主类型**：出海制造企业 / 本地集成商 / OEM（按脱敏口径描述）

## 2. 痛点 / 需求

- **触发事件**：出了什么问题 / 在建什么（例：新厂 PLC 缺人、既有产线视觉误检）
- **紧急程度**：为何必须现在解决（停产损失 / 投产节点 / 交付节点）
- **本地为何难解**：本地招聘周期、语言/时区、驻场排期等障碍（据实，不夸大）

## 3. 匹配与交付过程

- **方向（track）**：PLC / 机器人 / 视觉 / 电气
- **工程师地区**：（脱敏到大区）
- **匹配耗时**：从发布需求到指派持证工程师所用的**小时数**（据实记录，不四舍五入成营销数字）
- **认证等级**：被指派工程师在该方向的认证级别（L1 / L2 / L3）
- **用工形态**：远程 / 驻场 / 混合

## 4. 里程碑与金额

- **里程碑数量**：本单拆分为几个里程碑
- **金额区间**：交付总额落在哪个区间（用区间披露，不披露精确金额，如 "$X–Y k"）
- **平台费档位**：founding 5% / 标准 15%
- **托管 / 放款**：是否全程走里程碑托管、验收后放款？☐ 是

### 采集提醒

- 数字（匹配耗时、里程碑数、金额区间）一律**据实**，与平台后台记录核对一致。

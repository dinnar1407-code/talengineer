# PMF 外联名单 · 中国出海制造企业线（leads-cn）

> 目标客群：正在墨西哥 / 越南 / 泰国建厂或已公开宣布建厂计划的中国制造企业。
> 痛点假设：海外新厂投产时缺当地工业自动化工程师——本地招聘周期长、薪资高、语言与时区不通、驻场调试排期难。
> TalEngineer 价值主张（与 talengineer.us 一致）：*Automation talent without borders. Verified by AI. Protected by escrow.* —— AI 实测筛选 + 平台认证工程师，托管付款按里程碑放款，9 语言实时翻译，平均 48 小时匹配。
>
> **依据纪律**：每条均带公开新闻 / 园区名录 / 行业报告依据链接（见每行"依据"列），无依据不列。规模/位置为公开信息，标注"计划"或"已建/量产"以区分成熟度。

## 优先级说明（可触达性排序）

- **P1（最优先）**：A 股 / 港股上市 + 已在当地量产或已签约建厂 → 用人需求即时、有 IR/董秘公开联系渠道、决策链清晰。
- **P2**：整车/大厂已宣布建厂但仍在选址或早期 → 需求未来 6–12 月释放，先建关系。
- **P3**：已公开表达投资意向、尚未落地 → 培育型线索。

联系渠道均为**公开可触达入口**（官网/领英/投资者关系/出海园区招商），非已验证的私人联系方式；实际外联需二次核实对接人。

## 触达执行建议

1. **入口优先级**：A/港股公司 → 投资者关系/董秘邮箱（公开披露，回复率高于官网表单）；未上市 → 领英搜"公司名 + Mexico/Vietnam/Thailand + Plant/Operations/Automation"定位工厂负责人。
2. **出海园区批量入口**：Hofusan（蒙特雷）、Vynmsa（Nuevo León）、WHA/罗勇（泰国）、VSIP（越南）——园区招商方掌握新入驻名单，可作 BD 中转。
3. **展会**：广交会、Automate（美国）、宁波/深圳出海峰会、泰国 BOI 招商会——现场触达工厂筹建团队。
4. **话术要点**：新厂"投产爬坡期"缺当地 PLC/机器人/电气调试工程师 → TalEngineer 托管付款按里程碑放款（零预付风险）+ 9 语言实时翻译（解决语言/时区），平均 48 小时匹配持证工程师。
